Компактный дозиметр с низковольтным питанием.

Сделано максимально минималистично. В связи с низкой точностью одного счётчика гейгера, было решено отказатся от лишней электроники.

Питание 1 элемент R6 1,5в.
Звуковой сигнализатор. 
Стрелочная индикация.

Прибор имеет дополнительную УФ подсветку для любителей уранового стекла и просто проверки денег, что делает прибор чуть более полезным в быту.

Дополнительной функцией есть вольтметр показывающий заряд батареи.

![](https://raw.githubusercontent.com/74ls00/Geiger/master/pic/plata-top.jpg)

Схема питания собрана из деталей от китайского модуля 1,5в-5в c разъёмом usb предназначенная для зарядки телефона от одной батарейки.
Вместо трансформатора используется сетевой дроссель. Одна из обмоток частично смотана для получпния той же индуктивности что и оригинальный дроссель.
Вторая обмотка смотана полностью и намотана тонким проводом до получения около 440в как у ДП-5Б. 
В схеме используется каскад удвоения напряжения. При почти полном заполнении обмотки, напряжения оказалось недостаточно.

Напряжение замерялось через мультиметр DT9208 с входным сопротивление 10Мом и последовательно включенным ему набором резисторов 90Мом, показания прибора делились на 10.
После из за выхода из строя прибора, было замеряно простым DT838 с входным сопротивлением 1Мом.
